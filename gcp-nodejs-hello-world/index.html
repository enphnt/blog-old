node.js on GCP

So, it's the first step in any new development environment: HELLO WORLD! 🙌

Deploying a node.js app on GCP standard environment is actually incredibly easy.

Prereq steps:

  1. install and initiallize gcloud
  2. login to gcloud
  3. have billing enabled on your gcloud account 🤑
  4. have a node.js app that runs :)
  5. enable the Cloud Build API

Steps to Deploy:

  1. Create the Project - via the terminal with gcloud app create --project=[PROJECT_ID]
  2. Go to the node.js code directory or clone the google example: git clone https://github.com/GoogleCloudPlatform/nodejs-docs-samples
  3. Navigate to the Hello World app: cd nodejs-docs-samples/appengine/hello-world/standard
  4. Install and Run the app locally: npm install && npm start
  5. Deploy the app! - from the same root directory of the app, run: gcloud app deploy then follow the prompts to select region, etc.
  6. View the app once it finished deploying: gcloud app browse
Note: The default location for a deployment is: 
https://PROJECT_ID.REGION_ID.r.appspot.com
Alternatively, you can use: 
gcloud app browse
to have the page open for you.

And you are done! Yes, it was actually that easy. Now, lets look into some additional features/capabilities of GCP + Node.js, like GCP Datastore.
